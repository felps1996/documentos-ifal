# Sistema de Busca para Documentos Públicos

Projeto desenvolvido durante o PIBIC IFAL com o objetivo de criar um sistema de indexação e busca de documentos estaduais e federais.

## Minha participação
- Desenvolvimento de funcionalidades relacionadas à busca de dados
- Utilização de Elasticsearch para indexação e recuperação de informações
- Testes de APIs utilizando Postman
- Organização e estruturação de dados

## Tecnologias utilizadas
- JavaScript
- Python
- PHP
- Blade
- CSS
- Elasticsearch
- Postman

## Observação
Este repositório é um fork do projeto original, utilizado para fins de estudo e contribuição.
